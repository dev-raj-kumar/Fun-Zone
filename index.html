Projects(games) based upon DSA and Javascript

Sorting Visualizer

(DSA, HTML, CSS, Javascript, React.js)

• A React application for visualizing sorting algorithms
• Implemented Merge Sort, Quick Sort, Bubble Sort and Heap Sort
• It randomly generates an array of a given size by the user
and shows its sorting in varying speeds based upon size
• UI has been built by div and animations.

Sudoku Challenger

(DSA, HTML, CSS, Javascript)

• A javaScript application which gives sudoku of different difficulties to solve
• user can choose between easy, medium, hard and random, and fill the empty cells
• feature for verifying and getting the solution based upon the backtracking algorithm
• UI has been built with HTML div tag

Snake game

(DSA, HTML, CSS, Javascript)

• A traditional Snake game in which each time the snake eats the food, the score increment by one
• The speed of the snake increases gradually as your point increases.
• When the snake collides with the boundary wall game is over
• It is implemented with array as Data Structure and the UI is build over HTML canvas.
